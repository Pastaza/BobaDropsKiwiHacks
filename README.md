# Cloud Atlas ☁

An immersive 3D educational website about clouds, weather formation, and atmospheric layers. Built with React, Three.js, Framer Motion, and Tailwind CSS.

## Features

- 🌥 **Interactive 3D Cloud Scene** — Real-time WebGL cloud visualization with mouse parallax
- 📚 **Cloud Classification** — Learn about Cirrus, Cumulus, Stratus, Cumulonimbus, and Altostratus clouds
- 🌍 **Atmospheric Layers** — Explore the Troposphere, Stratosphere, Mesosphere, and Thermosphere
- 💧 **Cloud Formation** — Step-by-step visualization of how clouds form
- 🌡 **Climate Impact** — Discover how clouds regulate Earth's energy balance
- ✨ **Cinematic Design** — Glassmorphism UI, particle effects, and scroll-driven animations

## Tech Stack

- **Vite** — Fast build tooling
- **React 18** + **TypeScript** — Component framework
- **Three.js** + **@react-three/fiber** + **@react-three/drei** — 3D graphics
- **Framer Motion** — Animations and transitions
- **Tailwind CSS v3** — Utility-first styling

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

The dev server runs at http://localhost:5173
